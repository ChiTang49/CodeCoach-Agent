## 🎯 设计原则

1. **一致性**：所有页面使用统一的配色和组件风格
2. **可读性**：充足的留白和行高，易于阅读
3. **响应式**：适配不同屏幕尺寸
4. **交互反馈**：悬停、点击都有视觉反馈
5. **渐进式**：重要信息优先展示，细节可折叠

## 🔧 自定义建议

如需进一步定制，可修改 `app.py` 中的 `<style>` 标签部分：

1. 修改主题色：搜索 `#667eea` 和 `#764ba2` 替换
2. 调整圆角：修改 `border-radius` 值
3. 改变阴影：修改 `box-shadow` 参数
4. 更换字体：添加 `font-family` 引用
